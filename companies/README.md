Companies — Fundamentals Evaluator
=================================

This folder contains the architecture, starter code, and data layout for a company fundamentals evaluation tool.

Goals
- Ingest financial data for targeted companies
- Compute normalized fundamental metrics
- Produce a repeatable score and ranking for companies
- Be extensible (new metrics, backtesting, pipelines)

See `architecture.md` for design details and `src/evaluator.py` for a starter evaluator script.
