Research folder
===============

Drop research papers (PDF, TXT) here for automated idea extraction.

Guidelines
---------
- File naming: `YYYY-MM-DD_author_title.pdf` or similar for easy sorting.
- Preferred formats: `pdf`, `txt`, `md`.
- If uploading multiple files for the same paper, put them in a subfolder named after the paper.

What I'll do next
-----------------
- Extract text from uploaded papers.
- Summarize main ideas, methods, datasets, and results.
- Generate bullet-point action items and follow-up questions.

Example
------
`2026-03-29_smith_one-shot_learning.pdf`

Notes
-----
Keep files under 200MB where possible. If you upload large datasets or many PDFs at once, let me know so I can process them in batches.
