🚀 QUICK START (5 MINUTES)
════════════════════════════════════════════════════════════════════════════

1. Install Dependencies
   cd trading_system && pip3 install -r requirements.txt

2. Start Server
   python3 main_tradingview.py --tradingview --mode paper --webhook-port 5000

3. Create TradingView Alert
   • Chart: Any stock, any timeframe
   • Condition: close > open (or your strategy)
   • Webhook: http://YOUR_IP:5000/webhook/tradingview
   • Message: {"symbol":"{{ticker}}","close":{{close}},"high":{{high}},"low":{{low}},"open":{{open}},"volume":{{volume}},"time":"{{time}}","interval":"{{interval}}"}

4. Test Connection
   curl -X POST http://localhost:5000/webhook/tradingview \
     -H "Content-Type: application/json" \
     -d '{"symbol":"AAPL","close":150.25,"high":151,"low":149.5,"open":149.75,"volume":1000000,"time":"2026-03-19","interval":"5min"}'

5. Monitor
   tail -f trading_system.log

📊 WEBHOOK ENDPOINTS
════════════════════════════════════════════════════════════════════════════

POST   /webhook/tradingview
   • Receives TradingView alerts
   • Expects JSON with symbol, close, high, low, open, volume, time, interval
   • Returns: {"status": "success", "symbol": "AAPL", ...}

GET    /tradingview/status
   • Server status and metrics
   • Shows alerts_received, symbols_tracked, latest prices

GET    /tradingview/candles/<symbol>?count=20
   • Recent candles for symbol
   • Returns: Array of OHLCV data

POST   /tradingview/test
   • Test webhook with sample data
   • Useful for debugging

📋 TRADINGVIEW ALERT CONFIGURATION
════════════════════════════════════════════════════════════════════════════

1. Open TradingView chart (e.g., AAPL 5min)
2. Click Alerts → Create Alert
3. Set Condition:
   • Simple: close > open
   • Momentum: close > ta.sma(close, 20)
   • Multi: ta.macd(close) > ta.signal(close)
4. Set Webhook URL: http://YOUR_IP:5000/webhook/tradingview
5. Set Message (copy exactly):
   {
     "symbol": "{{ticker}}",
     "close": {{close}},
     "high": {{high}},
     "low": {{low}},
     "open": {{open}},
     "volume": {{volume}},
     "time": "{{time}}",
     "interval": "{{interval}}"
   }
6. Set Frequency: "On every alert trigger"
7. Enable notification (optional)
8. Create Alert

✓ Done! Your alert is now connected to your trading algorithm
